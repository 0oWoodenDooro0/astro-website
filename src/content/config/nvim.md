# Neovim Config

This is my complete Neovim configuration. It is built using Lua and follows a modular structure.

Key features:
- **LSP:** Native LSP configuration for multiple languages.
- **Treesitter:** Advanced syntax highlighting and code navigation.
- **Telescope:** Powerful fuzzy finder.
- **Lazy.nvim:** Fast and efficient plugin manager.
